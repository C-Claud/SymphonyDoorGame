# SymphonyDoorGame
Symphony Door Game – A classical music quiz where you guess symphonies from short clips. Features a built-in player with progress bar and three playback modes, one-click startup, and instant colored feedback. Fun and effortless music discovery.

# How to play
1. Put song files named properly into the dataset path
2. Make sure a Python 3 environment is ready
3. Click start.bat (if using Windows) or start.sh (if using MacOS or Linux)
